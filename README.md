# SharePoint 2013 - Migración de soluciones de Servidor a Addins

Microsoft SharePoint 2013 trajo consigo un nuevo paradigma de desarrollo de aplicaciones, los Addins. Este modelo se estableció, además, como la única manera de desarrollar aplicaciones en Microsoft SharePoint Online de Office 365. Todo esto conllevaba el reto de migrar nuestras antiguas aplicaciones si queríamos que siguiesen funcionando en las nuevas plataformas.
En este workshop, eminentemente práctico, explicaremos las diferencias entre ambos modelos de desarrollo y la mejor forma de migrar nuestras antiguas aplicaciones.
 
* Modelo de Addins vs Servidor
* Arquitectura técnica de la solución de Servidor
* Adaptación del código SSOM a CSOM
